# @rook/web

React + Vite frontend for The Rook Room.

## Scripts
Run from repo root unless noted.

```bash
# dev
npm run -w @rook/web dev

# build
npm run -w @rook/web build

# typecheck
npm run -w @rook/web typecheck

# lint
npm run -w @rook/web lint

# tests
npm run -w @rook/web test
```

For full project setup and game flow details, see the root `README.md`.
